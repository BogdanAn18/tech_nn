## 🧩 Что внутри

### 📦 Бэкенд (Express + PostgreSQL)
- Регистрация и вход с JWT
- Полное CRUD для подписок
- База данных с пользователями и подписками

### 🌐 Фронтенд (Next.js)
- Авторизация
- Список подписок
- Аналитика с графиками и прогнозом на год

### 📱 Мобилка (React Native + Expo)
- Те же функции, что и в вебе
- Навигация, контекст, работа с API
- Живёт в отдельной папке, но дышит с одним сервером

## 🚀 Как запустить (чтобы заработало с первого раза)

### 1. Клонируем
```bash
git clone https://github.com/BogdanAn18/tech_nn.git
cd tech_nn
```

### 2. Бэкенд
```bash
cd backend
npm install
```
Создай файл `.env` в папке `backend`:
```
JWT_SECRET=your-secret-key
DB_PASSWORD=твой-пароль-от-postgres
```
Запусти:
```bash
npm run dev
```
Сервер проснётся на `http://localhost:5000`

### 3. Фронтенд
```bash
cd ../frontend
npm install
npm run dev
```
Открывай `http://localhost:3000`

### 4. Мобилка
```bash
cd ../device_new
npm install
npm start
```
Скачай **Expo Go** на телефон, отсканируй QR-код — и приложение у тебя в кармане.
