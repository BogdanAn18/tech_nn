# 💸 Монитор подписок

Решение для отборочного задания направления «Фулстэк» — платформа для управления личными подписками.  
Проект включает серверную часть, веб-интерфейс и мобильное приложение, объединённые общим API.

---

## 🧩 Состав проекта

| Компонент       | Технологии                          | Описание |
|----------------|--------------------------------------|----------|
| **Бэкенд**      | Node.js, Express, PostgreSQL, JWT   | REST API с авторизацией и CRUD для подписок |
| **Фронтенд**    | Next.js, Recharts, CSS-модули       | Веб-интерфейс с аналитикой и графиками |
| **Мобильное приложение** | React Native, Expo, VictoryNative | Нативное приложение для iOS/Android |

---

## 🚀 Быстрый старт

### 1. Клонирование
```bash
git clone https://github.com/BogdanAn18/tech_nn.git
cd tech_nn
```

### 2. Бэкенд
```bash
cd backend
npm install
```
Создайте файл `.env`:
```env
JWT_SECRET=your-secret-key
DB_PASSWORD=your-postgres-password
```
Запуск:
```bash
npm run dev
```
Сервер будет доступен на `http://localhost:5000`

### 3. Фронтенд
```bash
cd ../frontend
npm install
npm run dev
```
Откройте `http://localhost:3000`

### 4. Мобильное приложение
```bash
cd ../device_new
npm install
npm start
```
Для запуска на устройстве установите **Expo Go** и отсканируйте QR-код.

## ✅ Функциональность

- Регистрация и вход с JWT-токенами
- Добавление, редактирование и удаление подписок
- Указание периода списания (дни, месяцы, годы)
- Аналитика: прогноз расходов на месяц/квартал/год
- Круговая диаграмма распределения по категориям
- Полная синхронизация между веб и мобильной версией

---

## ⚙️ Требования к окружению

- Node.js 18+
- PostgreSQL 14+
- npm или yarn
- Expo CLI (для мобильной разработки)

---

## 📌 Важно

Перед первым запуском убедитесь, что в папке `backend` создан файл `.env` с необходимыми переменными.  
База данных PostgreSQL должна быть запущена, а в ней созданы таблицы (скрипты миграции находятся в папке `backend/db`).
